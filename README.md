# Wplace 맞춤 지점 감시기

Wplace에서 원하는 지점을 자유롭게 추가하고 실시간으로 모니터링할 수 있는 웹 기반 감시 도구입니다.

## 🌟 주요 기능

### 📍 맞춤형 지점 관리
- **무제한 지점 추가**: 감시하고 싶은 지점을 원하는 만큼 추가
- **타일 좌표 기반**: Wplace의 타일 좌표를 이용한 정확한 위치 지정
- **픽셀 단위 조정**: 픽셀 좌표와 확대 비율을 통해 작품을 정밀하게 정렬
- **미리보기 기능**: 추가 전 이미지를 미리 확인

### 🔄 자동 새로고침
- **사용자 정의 간격**: 원하는 새로고침 주기 설정 (초 단위)
- **실시간 모니터링**: 설정된 간격마다 자동으로 이미지 갱신
- **개별 이미지 다운로드**: 각 지점의 확대된 이미지를 PNG 파일로 저장

### 💾 데이터 관리
- **로컬 저장**: 브라우저 로컬 스토리지에 자동 저장
- **내보내기/불러오기**: JSON 파일로 설정을 백업하고 복원
- **수정 및 삭제**: 추가된 지점을 언제든지 편집하거나 제거

### 🎨 사용자 경험
- **다크모드 지원**: 라이트/다크 테마 전환
- **반응형 디자인**: 다양한 화면 크기에 최적화
- **바로가기 링크**: 각 지점에 Wplace 위치 링크 추가 가능

## 📖 사용 방법

### 1️⃣ 지점 추가하기

1. **"감시 지점 추가" 카드 클릭**
2. **타일 좌표 입력** (Tl X, Tl Y)
   - 타일 좌표를 알아내려면 [블루마블](https://gall.dcinside.com/mgallery/board/view/?id=wplacelive&no=1204&search_head=30&page=1) 계열의 플러그인이 필요합니다
3. **"이미지 미리보기" 버튼 클릭**
4. **픽셀 좌표 및 설정 입력**
   - Px X, Px Y: 작품의 픽셀 좌표
   - 확대 비율: 이미지 확대/축소 배율
   - 최소 보장 폭 (선택): 관심 영역의 최소 너비 지정
5. **지점 이름 입력**
6. **장소 바로가기 링크 입력 (선택)**
   - 형식: `https://wplace.live/?lat=...` (공유 버튼 사용 후 붙여넣기)
7. **"추가" 버튼 클릭**

### 2️⃣ 새로고침 설정

- 페이지 상단의 **"새로고침 간격(초)"** 입력란에서 원하는 주기를 설정
- 설정한 간격마다 모든 지점의 이미지가 자동으로 갱신됩니다

### 3️⃣ 데이터 관리

#### 내보내기
- 상단의 **"내보내기"** 버튼 클릭
- `wplace-monitor-save.json` 파일 다운로드

#### 불러오기
- 상단의 **"불러오기"** 버튼 클릭
- 이전에 저장한 JSON 파일 선택
- ⚠️ 기존 데이터가 모두 덮어씌워집니다

### 4️⃣ 지점 관리

각 지점 카드에서:
- **📥 다운로드**: ~~현재 보이는 확대된 이미지 저장~~ (현재로서는 새탭에서 원본 타일 이미지가 열리며 이를 저장할 수 있습니다.)
- **🔗 바로 가기**: Wplace에서 해당 위치로 이동 (링크 설정 시)
- **✏️ 수정**: 지점 정보 편집
- **🗑️ 삭제**: 지점 제거

## 🛠️ 기능 상세 설명

### 타일 좌표 시스템
Wplace는 타일 기반 이미지 시스템을 사용합니다. 각 타일은 1000x1000 픽셀 크기이며, 타일 좌표(Tl X, Tl Y)로 위치를 지정합니다.

### 픽셀 좌표 및 확대 비율
- **픽셀 좌표**: 타일 내부의 정확한 픽셀 위치 (0~999)
- **확대 비율**: 1.0이 원본 크기, 2.0은 2배 확대
- **최소 보장 폭**: 지정 시 해당 픽셀 너비가 항상 화면에 표시되도록 자동 조정

## ❗ 문제 해결

### 이미지가 안 뜨는 경우
- **원인**: 네트워크 오류 또는 짧은 시간에 많은 이미지를 불러올 때 Rate Limit 발생
- **해결**: 인터넷 연결 상태를 확인하거나 새로고침 간격을 늘리고 잠시 후 재시도
- **확인**: [서버 상태 페이지](https://wplace.instatus.com/)에서 서버 상태 점검

### 타일 이미지가 깨지거나 꼬인 경우
- 버그일 가능성이 있습니다
- 페이지 하단의 Discord 링크를 통해 버그 신고

### 좌표를 모르는 경우
- Wplace 오버레이 플러그인 사용 권장
- [블루마블 관련 글](https://gall.dcinside.com/mgallery/board/view/?id=wplacelive&no=1204&search_head=30&page=1) 참고
